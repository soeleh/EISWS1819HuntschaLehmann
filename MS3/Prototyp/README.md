4. Installationsdokumentation
Für das Anwendungssystem wurde sich für eine MongoDB entschieden, die über https://mlab.com gehostet wird. Die Anmeldedaten sind in der server.js Datei zu finden.

Repository herunterladen (https://github.com/soeleh/EISWS1819HuntschaLehmann.git)
node.js installieren (https://nodejs.org/en/download/)


4.1 Server
Das Verzeichnis MS3/Prototyp/server öffnen
Das Terminal/Die Konsole in diesem Verzeichnis öffnen
Den Befehl npm install ausführen (dies kann einige Minuten dauern)
Mit dem Befehl node server.js den Server starten
Mit einem beliebigen REST Client Requests an localhost:2019/ senden

4.2 Client
Die CalmRunter.apk auf das Android Endgerät laden und installieren
Im App Menü taucht nun die CalmRunter Applikation auf und kann ausgeführt werden.
